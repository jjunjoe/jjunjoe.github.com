Markdown新手指南

Feb 2nd, 2014 10:00 pm

1、About Markdown

下面是来自维基百科的解释：

Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）和亚伦·斯沃茨（Aaron Swartz）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。 Markdown同时还是一个由Gruber编写的Perl脚本：Markdown.pl。它把用markdown语法编写的内容转换成有效的、结构良好的XHTML或HTML内容，并将左尖括号(‘<’)和&号替换成它们各自的字符实体引用。它可以用作单独的脚本，Blosxom和Movable Type的插件又或者BBEdit的文本过滤器.Markdown也已经被其他人用Perl和别的编程语言重新实现，其中一个Perl模块放在了CPAN(Text::Markdown)上。它基于一个BSD风格的许可证分发并可以作为几个内容管理系统的插件。

Markdown 的优点如下：

1、纯文本，所以兼容性极强，可以用所有文本编辑器打开。
2、让你专注于文字而不是排版。
3、格式转换方便，Markdown 的文本你可以轻松转换为 html、电子书等。
4、Markdown 的标记语法有极好的可读性。

说了这么多，其实我也不明白是什么意思，之所以我要学习它是因为我用octpress在github上搭建的个人博客需要用到它（没错，就是你现在看到的这个站点的文章就是Markdown格式的），所以，下面我就仅仅列一下我写github的blog需要用的markdown的基本用法。

2、Markdown基本操作

1）标题

在 Markdown 中，你只需要在文本前面加上 # 即可，同理、你还可以增加二级标题、三级标题、四级标题、五级标题和六级标题，总共六级，几级标题只需要增加相应的几个 # 即可，标题字号相应降低。语法举例如下：
1、一级标题

# 一级标题

或者

一级标题

======

2、二级标题

## 二级标题

或者

二级标题

3、三级标题

### 三级标题

4、四级标题

#### 四级标题

5、五级标题

##### 五级标题

6、六级标题

###### 六级标题

注：# 和「一级标题」之间建议保留一个字符的空格，这是最标准的 Markdown 写法。

2) 列表

列表格式也很常用，在 Markdown 中，你只需要在文字前面加上 - 或者 * 就可以了，例如：

  • 文本1
  • 文本2
  • 文本3

的语法应该是如下
- 文本1
- 文本2
* 文本3

如果希望是有序表，那么文字前加序号1，2，3等就可以了。

  1. 文本1
  2. 文本2
  3. 文本3

其语法应该是如下
1. 文本1
2. 文本2
3. 文本3

注：-、1.和文本之间要保留一个字符的空格。

3）链接和图片

在 Markdown 中，插入链接不需要其他按钮，你只需要使用
[显示文本](链接地址)
这样的语法即可，例如：
[github](www.github.com)
显示如下链接
github

图片则是前面加一个 ! 即可
![Mou icon](http://mouapp.com/Mou_128.png)
显示如下图片

4) 引用

引用他人的文字，在 Markdown 中，只需要在你引用的文字前面加上 > 就好了，例如：

Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯……

的语法如下:

> Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯……

> 和文本之间要保留一个字符的空格。

5) 粗体和斜体

Markdown 的粗体和斜体，用两个 * 包含一段文本就是粗体的语法，用一个 * 包含一段文本就是斜体的语法。例如：
粗体
斜体

的语法如下：
**粗体**
*斜体*

6）表格

关于表格的例子很奇怪，我是用Mou编辑的，在Mou里显示正常，但发布出来后显示就不对劲了，参考使用的话，需要取消语法示例理的 \ 符号
表格举例如下:

First Header Second Header Third Header
Content Cell Content Cell Content Cell
Content Cell Content Cell Content Cell

语法如下：

First Header | Second Header | Third Header
———— | ————- | ————
Content Cell | Content Cell | Content Cell
Content Cell | Content Cell | Content Cell

头尾也可以加上竖线:

First Header Second Header Third Header
Content Cell Content Cell Content Cell
Content Cell Content Cell Content Cell

语法如下：

| First Header | Second Header | Third Header |
| ———— | ————- | ———— |
| Content Cell | Content Cell | Content Cell |
| Content Cell | Content Cell | Content Cell |

还可以指定表格对齐方式:

First Header Second Header Third Header
Left Center Right
Left Center Right

语法如下：

First Header | Second Header | Third Header
:———– | :———–: | ———–:
Left | Center | Right
Left | Center | Right

7）注脚

注脚示例:

That’s some text with a footnote.1

格式如下
That’s some text with a footnote.[^1]

[^1]: And that’s the footnote.

8）代码

代码示例：

print("hello,world");

格式如下：
四个空格或者一个table键后面接代码。

9）文字背景色

文字背景色示例如下，` `（波浪符号~下面的那个符号）中间填加文字：

文字背景色示例

格式如下：
`文字背景色示例`

10) Octopress blog 中加图片

先copy图片到 /source/images目录下，然后在blog中用

<img src="/images/pic_name.png">

来引用，图片名称中不能有空格（也许有方式可以引用带空格文件名吧，我没去深究）。
